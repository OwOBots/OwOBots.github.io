a basic tf2 server setup guide

Buy a vps

you can use linode

harden your server by following this guide

open the ports 27015 and 27005

install the following packages

on arch run 

pacman -Syy tmux nano
and enable mutilib

on ubuntu run 

sudo apt update && sudo apt install tmux

install steamcmd

on arch run 

pacman -Syy lib32-gcc-libs lib32-libcurl-gnutls

on ubuntu run 

sudo dpkg --add-architecture i386 && sudo apt update && sudo apt install lib32z1 libncurses5:i386 libbz2-1.0:i386 lib32gcc-s1 lib32stdc++6 libtinfo5:i386 libcurl3-gnutls:i386 libsdl2-2.0-0:i386

make a folder called tf2_server

run nano tf2_ds.txt

paste this in

login anonymous
force_install_dir ./tf2_server
app_update 232250 validate
quit

run 

steamcmd +runscript tf2_ds.txt

install sourcemod and metamod

download the latest version of sourcemod and metamod

extract the files to the tf2_server folder

do cd tf2_server/tf/addons/sourcemod/configs/

run nano admins_simple.ini

paste this in

"YOUR_STEAM_ID" "99:z"

use a website to find your steamid

run 

cd tf2_server/tf/cfg/

run 

nano maplist.txt

do the same thing as mapcycle.txt

use a server.cfg generator

paste the generated server.cfg in the cfg folder

run 

nano server.cfg
and change 
sv_disable_respawn_times
to 
mp_disable_respawn_times

test your server first before you do anything else by running 

./tf2_server/srcds_run -game tf +map ctf_2fort +maxplayers 24
if it works then you can move on

make a file called run_server.sh

paste this in

./tf2_server/srcds_run -game tf +map ctf_2fort +maxplayers 24

run 

chmod +x run_server.sh

run 

tmux

run 

./run_server.sh

Installing plugins

NOTE: some plugins are drag and drop so READ THE README before you do this

download the plugin you want

extract the files to the tf2_server/addons/sourcemod/plugins folder

restart the server

how to detach from tmux and come back

press ctrl+b then d to detach

run 

tmux attach
to come back

Recommended plugins

StAC - a anticheat

Basic Votekick Immunity - Makes admins Immune to Votekicks

rafradek's srcds optimizer - dramatically reduces the CPU usage of the server
